# ASTRYX Skill Library

This directory is the org skill library — the canonical home for reusable, committed skills.
Skills here are finished, tested capabilities that any ASTRYX agent can invoke by name.
RSI (recursive self-improvement) lands here: when the org learns a better method, it becomes a committed skill.
Each skill lives in its own subdirectory with a manifest, implementation, and usage examples.
Do not place work-in-progress or experimental code here; committed only.
